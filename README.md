GUI Calculator using Python

This project is a simple graphical calculator built using Python. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division through an easy-to-use graphical interface. The project focuses on understanding GUI development in Python and handling user input through buttons and events.
